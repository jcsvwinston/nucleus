## Goal

**ADR-010 Phase 3.1 — env-layer attribution + `file:line` provenance**
(started 2026-05-23). Complete the effective-config provenance picture
deferred from Phase 3a: make the env config layer real in the nucleus
`FromConfigFile` path (so `[env:NUCLEUS_*]` sources appear and env overrides
actually apply, honouring ADR-010 §4 `defaults < files < env`), and add
`file:line` provenance to file-sourced keys.

## Scope

- **Slice A — env layer (primary).** Wire the `NUCLEUS_`-prefixed env layer
  (koanf `env.Provider`, `__`→`.` transform, same as `app.LoadConfig`) into
  `loadMerged` AFTER the file loop. Attribute each env-overridden key as
  `ConfigSource{Kind:"env", Path:<ENV_VAR_NAME>}`. **Behavioural fix:** the
  nucleus path currently ignores env entirely (`app.New` only `mergeDefaults`,
  `loadFromFiles` never loads env) — so this is the first time env overrides
  take effect via the fluent builder, closing the ADR §4 precedence gap.
  - Env keys NOT in the schema are **ignored** (env is an ambient namespace;
    only recognised keys are config) — distinct from files, where unknown keys
    are authored mistakes worth flagging strictly. Decision pending
    architect/security review in-loop.
- **Slice B — `file:line` provenance.** Add `Line int` to `ConfigSource`
  (additive; `omitempty`). Capture the source line for file-sourced keys:
  YAML via `yaml.v3` `yaml.Node` walk; TOML via go-toml v2 positions (verify
  feasibility); JSON has no standard line API → kind+path only (documented).
- out: CLI flags layer + programmatic-override attribution (ADR §4 tail);
  `config schema` emission; relocating any stable surface.

## Acceptance criteria

- [ ] `nucleus config print --effective` / `GET /_/config` show `[env:NUCLEUS_*]`
      for env-overridden keys, and env values actually override file values
      (precedence `defaults < files < env`).
- [ ] Unknown `NUCLEUS_`-prefixed env vars do not pollute the typed config or
      the effective snapshot.
- [ ] File-sourced keys report `file:line` for YAML and TOML; JSON reports
      kind+path (no line).
- [ ] `ConfigSource.Line` added additively → freeze baseline +1 (additive
      rebaseline), no removed/renamed symbol.
- [ ] `go test ./...` green; iteration loop clean; CHANGELOG + docs + website
      updated.
