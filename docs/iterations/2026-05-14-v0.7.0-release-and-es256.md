   `go mod tidy` cannot run cleanly (pre-existing `admin/proto`
   replace-directive issue). Re-run once that is unblocked.
2. **Phase 4 — AWS SDK opt-in** — `pkg/app` imports `pkg/auth/secrets`
   unconditionally, so every binary pays the ~3-5 MB AWS SDK link cost.
   Consider a build tag or plugin pattern.

   `EncryptionKey` in production (audit §7 item 5; security gap).

   `authorization` / `cookie` / `password` / `token` / `secret` fields
   (audit §7 item 6).

   Oracle — the scaffolds currently have only string-match tests
   (audit §7 item 7).
6. **Schema-level drift detection** via `information_schema` introspection
   — the checksum drift landed this iteration is the file-level half
   (audit §7 item 8).
7. **503 path test for `/healthz`** (audit §7 item 9).
8. **Future secret-manager resolvers** — GCP Secret Manager, Azure Key
   Vault, HashiCorp Vault. Explicit non-goals of ADR-005; the
   `secrets.Resolver` interface is the seam.
9. **`tasks.Manager` struct→interface** — contract-guardian advised an
